Api testing tool

An API testing tool I built to simplify creating, running and automating HTTP tests for RESTful and GraphQL services. It supports saved requests, environment variables, chaining requests, automated assertions, and generates detailed request/response logs for debugging. I implemented a clean UI for composing requests, a lightweight runner for assertions, and exportable JSON test suites so tests can be versioned and integrated into CI pipelines.

Meeto.ai — Google Meet extension

Meeto.ai is a Chrome extension I developed for Google Meet that analyzes meetings in real time. It provides live transcription, speaker detection, sentiment and keyword highlights, and automatic action-item extraction so teams can focus on conversation instead of note-taking. Designed with privacy controls, Meeto.ai processes audio securely (client-first for lightweight features; optional server-side NLP for advanced summaries) and exports concise meeting summaries and timestamped notes for easy sharing.
